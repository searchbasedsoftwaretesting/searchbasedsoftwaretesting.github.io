Tool Competition

This year as well we are pleased to announce the eighth edition of the unit testing tool competition. We invite researchers to participate in the competition with their unit test generation tool for Java. Tools will be evaluated against a benchmark with respect to code coverage and mutation score.

To participate:
  • notify the organizers by sending an email to Xavier Devroey, Sebastiano Panichela, and Alessio Gambi (x.d.m.devroey@tudelft.nl, panichella@ifi.uzh.ch, alessio.gambi@uni-passau.de).
  • check out the contest infrastructure and make sure your tool works well within the infrastructure.

The competition infrastructure is available on GitHub and can also be tried using Docker. Details can be found at https://github.com/JUnitContest/junitcontest.

Important dates

  • Tool submission: February 3
  • Benchmark results communicated to authors: February 28
  • Submission of camera-ready paper: March 16 April 7
